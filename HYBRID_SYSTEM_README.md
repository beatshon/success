# 🔗 하이브리드 트레이딩 시스템

뉴스 분석과 기술적 분석을 결합한 종합 투자 분석 시스템입니다.

## 🎯 시스템 개요

### 핵심 특징
- **뉴스 분석**: 네이버 뉴스 API를 통한 실시간 뉴스 수집 및 감정 분석
- **기술적 분석**: RSI, MACD, 이동평균, 거래량 등 기술적 지표 분석
- **하이브리드 결합**: 뉴스 분석(40%) + 기술적 분석(60%) 가중 평균
- **실시간 대시보드**: 웹 기반 실시간 모니터링 시스템

### 분석 지표
1. **뉴스 분석 점수** (0-100)
   - 긍정/부정 감정 분석
   - 종목 관련성 매칭
   - AI 기반 키워드 매칭

2. **기술적 분석 점수** (0-100)
   - RSI (상대강도지수)
   - MACD (이동평균수렴확산)
   - 이동평균 (5일/20일)
   - 거래량 분석
   - 지지/저항선

3. **종합 점수** (0-100)
   - 뉴스 점수 × 0.4 + 기술적 점수 × 0.6

## 🚀 빠른 시작

### 1. 환경 설정
```bash
# 가상환경 활성화
source news_env/bin/activate

# 필요한 패키지 설치
pip install -r requirements.txt
```

### 2. API 키 설정
```bash
# 네이버 API 키 설정
python update_api_keys.py
```

### 3. 시스템 실행

#### 전체 시스템 실행 (분석 + 대시보드)
```bash
python run_hybrid_system.py --mode full
```

#### 분석만 실행
```bash
python run_hybrid_system.py --mode analysis
```

#### 대시보드만 실행
```bash
python run_hybrid_system.py --mode dashboard
```

#### 상태 확인
```bash
python run_hybrid_system.py --mode status
```

## 🔍 분석 결과

### 매매 신호 분류
- **강력매수**: 종합점수 80점 이상
- **매수**: 종합점수 60-79점
- **관망**: 종합점수 41-59점
- **매도**: 종합점수 21-40점
- **강력매도**: 종합점수 20점 이하

### 출력 파일
1. **CSV 파일**: `data/hybrid_analysis/hybrid_analysis_YYYYMMDD_HHMMSS.csv`
   - 종목별 상세 분석 결과
   - 뉴스 점수, 기술적 점수, 종합 점수
   - 매매 신호 및 추론 과정

2. **리포트 파일**: `data/hybrid_analysis/hybrid_report_YYYYMMDD_HHMMSS.txt`
   - 매수/매도 추천 종목 요약
   - 섹터별 분석 결과

## 🔧 기술적 지표

### RSI (상대강도지수)
- **과매수**: 70 이상
- **과매도**: 30 이하
- **중립**: 40-60

### MACD
- **매수 신호**: MACD가 시그널선을 상향돌파
- **매도 신호**: MACD가 시그널선을 하향돌파

### 이동평균
- **골든크로스**: 5일선이 20일선을 상향돌파
- **데드크로스**: 5일선이 20일선을 하향돌파

### 거래량
- **높음**: 평균 대비 1.5배 이상
- **낮음**: 평균 대비 0.5배 이하
