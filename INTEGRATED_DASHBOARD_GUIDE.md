# 🎯 통합 대시보드 가이드

## 개요

통합 대시보드는 키움 트레이딩 시스템의 모든 구성 요소를 한 곳에서 모니터링하고 관리할 수 있는 중앙 허브입니다. 하이브리드 분석과 시뮬레이션 결과를 실시간으로 확인하고, 시스템 상태를 모니터링하며, 데이터를 내보낼 수 있습니다.

## 🚀 주요 기능

### 1. 실시간 모니터링
- **하이브리드 분석 결과**: 매수/매도/관망 신호 분포, 종목별 점수
- **시뮬레이션 성과**: 수익률, 승률, 샤프 비율, 최대 낙폭
- **시스템 상태**: 데이터 가용성, 업데이트 시간, 서비스 상태

### 2. 시각화 차트
- **포트폴리오 성과 트렌드**: 시간별 포트폴리오 가치 변화
- **매매 신호 분포**: 도넛 차트로 신호별 분포 표시
- **포트폴리오 성장률**: 일별 성장률 바 차트

### 3. 데이터 관리
- **데이터 검증**: 자동 데이터 품질 검사
- **알림 시스템**: 오류, 경고, 성과 알림
- **데이터 내보내기**: CSV 형식으로 데이터 다운로드

### 4. 실시간 업데이트
- 30초마다 자동 데이터 새로고침
- 실시간 차트 업데이트
- 동적 알림 표시

## 📊 대시보드 구성

### 메인 섹션
1. **개요 통계**
   - 하이브리드 분석: 총 종목, 매수/매도/관망 신호 수
   - 시뮬레이션 성과: 총 수익률, 승률, 총 거래, 최대 낙폭, 샤프 비율

2. **알림 섹션**
   - 시스템 오류 및 경고
   - 성과 달성 알림
   - 데이터 신선도 경고

3. **상세 정보**
   - 상위 성과 종목
   - 최근 활동 로그
   - 섹터 분포
   - 최근 거래 내역

4. **차트 섹션**
   - 포트폴리오 성과 트렌드
   - 매매 신호 분포
   - 포트폴리오 성장률

5. **시스템 상태**
   - 데이터 가용성 상태
   - 포트폴리오 성장률 차트

## 🔧 설치 및 실행

### 1. 의존성 설치
```bash
pip install flask pandas loguru
```

### 2. 대시보드 실행
```bash
python integrated_dashboard.py
```

### 3. 접속
브라우저에서 `http://localhost:8080` 접속

## 📈 API 엔드포인트

### 데이터 조회
- `GET /api/overview` - 개요 데이터
- `GET /api/quick-stats` - 빠른 통계
- `GET /api/recent-activity` - 최근 활동
- `GET /api/system-status` - 시스템 상태

### 차트 데이터
- `GET /api/chart-data/performance-trend` - 성과 트렌드
- `GET /api/chart-data/signal-distribution` - 신호 분포
- `GET /api/chart-data/portfolio-growth` - 포트폴리오 성장

### 데이터 관리
- `GET /api/data-validation` - 데이터 검증
- `GET /api/alerts` - 알림 조회
- `GET /api/export/hybrid-data` - 하이브리드 데이터 내보내기
- `GET /api/export/simulation-data` - 시뮬레이션 데이터 내보내기

## 🔍 데이터 검증

- 필수 컬럼 존재 확인
- 데이터 타입 검증
- 신호 값 유효성 검사

- 데이터 타입 검증

## 🚨 알림 시스템

### 알림 유형
- **오류 (Error)**: 데이터 누락, 시스템 오류
- **경고 (Warning)**: 데이터 품질 문제, 신선도 경고
- **성공 (Success)**: 높은 수익률, 높은 승률 달성

### 알림 조건
- 데이터 파일 누락
- 데이터 품질 문제
- 수익률 10% 이상 달성
- 승률 70% 이상 달성
- 최대 낙폭 20% 이상 발생
- 데이터 1시간 이상 미업데이트

## 📊 성과 지표

### 수익률 지표
- **총 수익률**: 전체 기간 수익률
- **샤프 비율**: 위험 대비 수익률
- **최대 낙폭**: 최대 손실 폭

### 거래 지표
- **총 거래 수**: 전체 거래 횟수
- **승률**: 수익 거래 비율
- **평균 수익률**: 거래당 평균 수익률

## 🔄 자동화 기능

### 실시간 업데이트
- 30초마다 데이터 자동 새로고침
- 차트 실시간 업데이트
- 알림 자동 표시/숨김

### 데이터 동기화

## 🛠️ 문제 해결

### 일반적인 문제
1. **데이터가 표시되지 않음**
   - 데이터 디렉토리 확인
   - 파일 권한 확인
   - 로그 확인

2. **차트가 로드되지 않음**
   - Chart.js 라이브러리 확인
   - 브라우저 콘솔 오류 확인
   - 데이터 형식 확인

3. **알림이 표시되지 않음**
   - 데이터 검증 결과 확인
   - 알림 조건 확인
   - JavaScript 오류 확인

### 로그 확인
```bash
# 대시보드 로그 확인
tail -f logs/dashboard.log
```
