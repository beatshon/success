## 🚨 **예상 문제점 및 해결방안**

### **문제 1: 키움 API 연결 실패**
- **원인**: 키움 Open API+ 미설치 또는 설정 오류
- **해결**: 키움 Open API+ 재설치 및 설정 확인

### **문제 2: Mac-Windows 통신 실패**
- **원인**: 네트워크 연결 또는 방화벽 문제
- **해결**: 포트 8080 방화벽 설정 확인

### **문제 3: 실시간 데이터 수신 실패**
- **원인**: 장 휴무일로 인한 데이터 제한

### **문제 4: 주문 전송 실패**
